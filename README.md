# ws-scrcpy for Raspberry Pi

This repository provides a docker images for [Web client prototype for scrcpy](https://github.com/NetrisTV/ws-scrcpy) built for Raspberry Pi.

Images for the next platforms currently available on Docker Hub:
- linux/arm/v7
- linux/arm64
